# chatserver
可以工作在nginx tcp负载均衡环境中的集群聊天服务器和客户端源码 
# 开发环境
  1. ubuntu linux环境
  2. 安装Json开发库
  3. 安装boost + muduo网络库开发环境，参考我的博客：
  4. 安装redis环境
  5. 安装mysql数据库环境
  6. 安装nginx
  7. 安装CMake环境
## 项目安装
```
//安装目录下直接执行
./Autobuild.sh
```

# 实现功能
  1. 客户端新用户注册
  2. 客户端用户登录
  3. 添加好友和添加群组
  4. 好友聊天
  5. 群组聊天
  6. 离线消息
  7. nginx配置tcp负载均衡
  8. 集群聊天系统支持客户端跨服务器通信
